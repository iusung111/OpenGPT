# Task Timer

A standalone Python GUI project built with `tkinter`.

## Features

- Add tasks
- View the task list
- Start a 25-minute timer for the selected task
- Stop the timer
- Reset the timer
- Mark a task as complete

## Requirements

- Python 3.x
- No external packages

## Run

From this directory on Windows:

```bash
python main.py
```

## Notes

- The default timer duration is 25 minutes per task.
- Only one task timer runs at a time.
- Completed tasks are shown with a check mark.
